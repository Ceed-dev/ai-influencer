## 1. エグゼクティブサマリー

AIインフルエンサーアカウントを大量運用し、ショート動画（YouTube Shorts / TikTok / Instagram Reels / X）を自動制作・投稿・分析するシステム。2月に50アカウントでMVP開始、6月までに3,500アカウントへスケール。

収益源はプラットフォーム広告収益＋アフィリエイト。Node.jsパイプラインで動画を自動生成し、GASアナリティクスでパフォーマンス分析・改善提案を行う。

### 現状（2026-02-11時点）

- **パイプライン v4.0 完成**: 入力 → 動画自動生成 → Drive保存（~12分/本）が稼働
- **インベントリ実データ投入済み**: 7アカウント + 12 Gmail + シナリオ3本 + モーション3本
- **投稿は手動**: Driveに溜まった動画を人間が各プラットフォームに手動投稿
- **次のステップ**: fal.ai クレジット購入（$500、チーム承認待ち）→ 量産開始

### 直近の方針

> **コンテンツ（動画）を自動生成し、Google Driveに溜める。人間がそこから手動で投稿する。**

- **Shungo**: パイプライン構築・保守
- **チームメイト**: アカウント設計（50アカウントのペルソナ・シナリオ）、コンテンツ品質担保
- **インターン**: 手動投稿オペレーション
- **自動投稿・メトリクス収集・分析ループは全て後続フェーズ**

> 技術的な詳細（パイプライン構成、API仕様、データスキーマ等）は [README.md](../README.md) と [ARCHITECTURE.md](ARCHITECTURE.md) を参照。

## 3. 収益モデル

### 各プラットフォームの収益化条件

#### YouTube

**収益化条件:**
- 登録者 1,000人
- 直近90日で有効な公開アップロード 3本
- さらに以下のどちらか:
  - 過去12か月の公開の有効な総再生時間 4,000時間
  - 直近90日の有効な公開Shorts視聴数 1,000万回

**その他の条件:**
- YPP（YouTubeパートナープログラム）加入
- 2段階認証（2-Step Verification）ON
- AdSense for YouTube アカウントをリンク

**収益度合い:**
- 長尺動画: 200〜500円/1,000再生
- ショート: 3〜30円/1,000再生
- ※広告費をかける時期によって大幅に変動

**アフィリエイト案件単価:**
- 長尺動画: 20〜100万円
- Shorts: 5〜20万円

#### TikTok

**収益化条件:**
- フォロワー 1万人+
- 直近30日間の総動画視聴数 10万回以上
- 1,000以上の動画から収益化発生
- 5秒未満の視聴はカウント外
- おすすめFeed以外からの流入分の視聴はカウント外

**その他の条件:**
- 日本在住 & 18歳以上
- 動画の長さが1分以上であること
- 政府関連やビジネス等の非個人アカウントは不可

**収益度合い:**
- CPM: 40円〜100円

**アフィリエイト案件単価:** 10〜30万円

#### Instagram

- **プラットフォームからの収益は基本的に存在しない**（Metaからの招待制）
- **基本的にアフィリエイトでの収益がメイン**

**アフィリエイト案件単価:** 10〜50万円

#### X / Twitter

**収益化条件:**
- X Premium（有料）への加入
- フォロワー 500人以上
- 直近3ヶ月の総合インプレッション 500万回以上

**収益度合い:**
- CPM: 30〜100円

**アフィリエイト案件単価:** 5〜30万円

## 6. 全体フロー

### 現在のフロー

```
市場リサーチ → アカウント設計 → コンテンツ作成 → 投稿 → 計測 → 分析 → 改善 → ループ
    🔴             🟡             🟢          🟡     🟡      🔴     🔴
```

| フェーズ | 状況 | 備考 |
|---|---|---|
| 市場リサーチ | 🔴 未実装 | ClipPulse検討中 |
| アカウント設計 | 🟡 手動 | Sheets管理、50アカウント設計中 |
| コンテンツ作成 | 🟢 自動 | Pipeline v4.0（~12分/本） |
| 投稿 | 🟡 手動 | Driveから人間が投稿。Phase 2でAPI自動化予定 |
| 計測 | 🟡 半自動 | CSV手動エクスポート → GAS自動取込 |
| 分析 | 🔴 実装途中 | GAS + OpenAI GPT-4o（基盤実装済み、本番運用未開始） |
| 改善 | 🔴 実装途中 | コンポーネントスコア更新 → 次回推奨（基盤実装済み、本番運用未開始） |

🟢 自動　🟡 半自動/手動　🔴 未実装
